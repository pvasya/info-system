#### Інформаційна система

#### Інтернет-магазин

#### Використані інструменти та технології:
- Python
- Jinja
- Docker
- PostgreSQL
- postgres-exporter
- Blackbox
- blackbox-exporter
- Prometheus
- Grafana

#### Опис: Система Інтернет-магазин з розширеною функціональністю

Реалізовано різні типи доступу. Ролі: гість, користувач(покупець), адміністратор.

Також для тестування навантаження створено генератор даних.

postgres-exporter для моніторингу підключень, кешу, розміру таблиць

Blackbox для метрик сайту, час завантаження сторінок, розмір сторінок

#### Запуск

`docker compose up -d --build`

Нище огляд за допомогою знімків екрану
